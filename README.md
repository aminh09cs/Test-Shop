## Install the dependencies
```bash
yarn 
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
yarn dev
```
### Choose port
```bash
If the local port doesn't work, choose another port
```
### Lint the files
```bash
yarn lint
# or
npm run lint
```


### Format the files
```bash
yarn format

npm run format
```



### Build the app for production
```bash
quasar build
```
